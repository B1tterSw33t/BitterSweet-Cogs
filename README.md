## EMBEDSPEAKER
> Messages to Embeds

Simple tool to convert messages into embeds the moment they're sent.

## Commands
- `embedspeaker add`
    - Enable the channel the command was sent in.
- `embedspeaker remove`
    - Disable the channel the command was sent in.
